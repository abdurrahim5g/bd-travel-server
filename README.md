# Project Title

This is a basic express server. We have 3 different API for for connunicate fo backend.

## API Reference

#### Get all places

```http
  GET http://localhost:5000/places/
```

#### Get Single place

```http
  GET http://localhost:5000/place/${id}
```

### Get Hotels

```http
   GET http://localhost:5000/place/hotels/${id}
```
